# Spring Boot Oracle JPA Hibernate CRUD Application

This project is a simple CRUD (Create, Read, Update, Delete) application developed using:

- Spring Boot
- Spring Data JPA
- Hibernate
- Oracle Database
- Maven
- Java 17

The application connects Spring Boot with Oracle Database and performs database operations using JPA Repository.

---

# Technologies Used

| Technology | Version |
|---|---|
| Java | 17 |
| Spring Boot | 3.3.0 |
| Hibernate | 6 |
| Oracle Database | Oracle XE |
| Maven | Latest |

---

# Project Features

- Insert Employee
- Update Employee
- Delete Employee
- View All Employees
- Search Employee
- Oracle Database Connectivity
- JPA Repository Support
